# Remax Universe Template

使用 Remax 开发跨多端小程序的模板。

通过该模板创建一个新项目：

```bash
$ npx degit remaxjs/template-universe my-app
$ cd my-app
```

## 开始开发

安装依赖

```bash
$ npm install
```

开始构建

```bash
$ npm run dev wechat # 选择对应的平台，如 wechat，alipay
```

使用对应的小程序开发者工具打开项目下的 `dist` 目录。

## 发布

```bash
$ npm run build wechat # 选择对应的平台，如 wechat，alipay
```

使用对应小程序开发者工具上传版本。
