# Remax Wechat Template TypeScript

使用 Remax 开发微信小程序的 TypeScript 模板。

通过该模板创建一个新项目：

```bash
$ npx degit remaxjs/template-wechat-typescript my-app
$ cd my-app
```

## 开始开发

安装依赖

```bash
$ npm install
```

开始构建

```bash
$ npm run dev
```

使用微信小程序开发者工具打开项目目录。

## 发布

```bash
$ npm run build
```

使用微信小程序开发者工具上传版本。
